# Agile Scrum Presentation Deck

Interactive presentation about Agile practices and Scrum, built as a self-contained HTML slide deck.

This project was created to support executive and operational conversations about agility, especially in organizations that do not develop software but still need better visibility, prioritization and delivery cadence.

The presentation explains:

• The operational problems caused by lack of work visibility  
• A simple agile practice usable across departments  
• The executive value of structured work cadence  
• The foundations of Scrum  
• Roles, artifacts and events  

The deck is implemented as a single HTML file with embedded styles and scripts, making it easy to run, share or modify.
